FOLDERS
>map		- Compressed Maps (with file extension *.map)
>data		- Map Data (Auto-generated)
>fld		- Uncompressed KA-Pro Maps (with file extension *.fld)
>colored_maps	- Colored Maps (Formats accepted: BMP,JPEG,GIF,PNG,PSD,XBM,PCD,PCX,RAS,XPM)

If some maps are missing in MAPS folder, you can copy new maps on FLD folder based from other Kore Maps.

Example:
If unknown.map is missing in "map" folder, you can use unknown.fld and place it in "fld" folder.
